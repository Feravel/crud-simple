# CRUD Test
### Intro
Hice un CRUD básico en el que a partir de dos input (Título y Descripción) se le puede añadir datos nuevos a la base de datos. Estos datos se pueden borrar, marcar como completados y/o editar la descripción.

### Qué usé
* HTML/CSS
* Tailwind / Bootstrap
*	Node.js
*	Express.js
*	SQLite
* Thunder Client (para testear POST/PUT/DELETE)
